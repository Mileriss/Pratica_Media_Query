O que é Media Query?

Media Query é utilizado para criar "regras" de formatações no estilo de uma página HTML. É possível criar determinadas regras para que a página siga uma formatação especifica a partir de uma determinada condição.

Essas regras são muito importantes para uma formatação responsiva de uma página, afim de proporcionar a visualização correta de uma página em todos os tipos de dispositivos (Celulares, Tablets, Pcs e etc...)

Por exemplo, é possível criar uma página que será exibida em uma tela de computador e posteriormente criar regras com Media Queries na qual essa exibição será ajustada ao tamanho exato de um celular.

Media Types

Permite definir para qual tipo de formatação as regras deverão ser aplicadas.
Os tipos disponíveis são:

  • Screen

    • - Para formatações realizadas em Telas

  • Print

    • - Para formatações realizadas em Impressões

  • Speech

    • - Para formatações realizadas em Sintetizadores de voz

  • All

    • - Para formatações realizadas em todos mencionados acima

Operadores Lógicos

Por se tratar de regras de formatação, a funcionalidade Media Query permite que você crie condições especificas para chegar no resultado desejado.

Formatações individuais:

@media screen (min-width: 400px){} -> A formatação será aplicada apenas se a tela estiver no mínimo com 400 píxels

Formatações com operadores (AND):

@media screen (min-width: 400px) and (max-width: 500px) {} -> A formatação será aplicada se a tela estiver entre 400px e 500px

Formatações com operadores (OR):

@media screen (min-width: 400px),(max-width-500px) {} -> A formatação será aplicada se a tela estiver com no mínimo 400 píxels OU no máximo com 500 píxels

Exemplos de uso das Media Queries

A cor deste bloco será vermelha caso ele tenha MENOS de 500px

A cor deste bloco será azul caso ele tenha MAIS de 500px

A cor deste bloco será roxa caso ele esteja ENTRE 400px e 500px

A cor deste bloco será verde caso ele tenha no minimo 500px OU no máximo 400px
